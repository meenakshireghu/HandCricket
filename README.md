# HandCricket
Python Hand Cricket Game
## Overview
The **Hand Cricket Game** is a simple, interactive Python game where a user competes against the computer. The game follows the basic rules of hand cricket with a toss, batting, bowling, and scoring system.

## Features
* Toss to determine who bats or bowls first
* Interactive gameplay with user and computer taking turns
* Simple run scoring and out mechanism
* Fun and strategic gameplay with a random number generator
